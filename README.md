# 📊 Gerador de Tabuada

Este é um site simples e interativo para **gerar a tabuada de qualquer número**.
O objetivo do projeto é ajudar estudantes a **praticar multiplicação de forma rápida e visual**, além de servir como exercício de desenvolvimento web.

🔗 **Acesse o site:**
https://site-tabuada.netlify.app/

---

## 🚀 Funcionalidades

* 🔢 Gerar a tabuada de qualquer número digitado
* ⚡ Resultado exibido instantaneamente na tela
* 📱 Interface simples e fácil de usar
* 🎓 Ideal para estudantes que estão aprendendo multiplicação

A tabuada é uma ferramenta clássica utilizada no ensino da matemática para facilitar cálculos e o aprendizado das operações básicas, especialmente a multiplicação. Multiplicação ([Tabuada][1])

---

## 🖥️ Tecnologias utilizadas

Este projeto foi desenvolvido utilizando:

* **HTML5** – Estrutura da página
* **CSS3** – Estilização e layout
* **JavaScript** – Lógica para gerar a tabuada dinamicamente

---

## 📷 Demonstração

Digite um número no campo indicado e clique no botão para gerar automaticamente a tabuada correspondente.

Exemplo:

```
Tabuada do 5

5 x 1 = 5
5 x 2 = 10
5 x 3 = 15
5 x 4 = 20
5 x 5 = 25
5 x 6 = 30
5 x 7 = 35
5 x 8 = 40
5 x 9 = 45
5 x 10 = 50
```
